# MedNova OS Web Dashboard

## Run on your Mac

1. Extract this folder into `~/MedNova-OS`.
2. Open Terminal and run:

```bash
cd ~/MedNova-OS/mednova_os_web
python3 -m pip install -r requirements.txt
python3 app.py
```

3. Open `http://127.0.0.1:5000`.

The app automatically uses `~/MedNova-OS/database/nafdac_intelligence.db`.
